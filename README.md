# Open Data Gacha

Open Data Gacha is a digital collectible card game, using the datasets from the Berlin Open Data Portal. The cards are available in different rarities and with different attack ATK) and defense (DEF) stats to show the quality of their underlying datasets. 

DISCLAIMER: This project was vibecoded with [v0](https://v0.app/) and was strongly inspired by [Wikipedia Gacha](https://wikigacha.com).
